# 🌐 Deploy on GitHub Pages

## Push Code

```bash
git init
git add .
git commit -m "Initial Commit"
git branch -M main
git remote add origin https://github.com/yourusername/luxora-interior-website.git
git push -u origin main
```

---

## Enable GitHub Pages

Go to:

```bash
Settings → Pages
```

Then:

- Source → Deploy from branch
- Branch → main
- Folder → /root
- Save

Your site will be live at:

```bash
https://yourusername.github.io/luxora-interior-website/
```
